# New Paper Imported — A Methodology For Selecting And Composing Runtime Architecture Patterns For Prod

*Imported: 2026-05-21 11:05*

## Summary

**What it is:** The stochastic-deterministic boundary (SDB) is a proposed concept that specifies how a large language model (LLM) output becomes a system action in production LLM agents.
**Key contribution:** The SDB is defined as a four-part contract among a proposer, a verifier, a commit step, and a reject signal, which provides a framework for designing and composing runtime architecture patterns for production LLM agents.
**Main finding:** The proposed methodology for selecting and composing runtime architecture patterns using the SDB concept is validated through a five-step selection methodology and a diagnostic procedure, which separates two contributions to long-run agent reliability: per-call variance from the stochastic proposer and architectural momentum set by pattern choice and SDB strength.
