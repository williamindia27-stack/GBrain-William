# New Paper Imported — A Methodology For Selecting And Composing Runtime Architecture Patterns For Prod

*Imported: 2026-05-20 16:29*

## Summary

**What it is:** The stochastic-deterministic boundary (SDB) is a four-part contract that specifies how a large language model (LLM) output becomes a system action in production LLM agents.
**Key contribution:** The proposed methodology organizes three orthogonal concerns for production agent runtimes and provides a catalog of six patterns that assemble the SDB differently across runtime classes, allowing for explicit design around the boundary.
**Main finding:** The application of the methodology to five workloads demonstrates that architectural momentum, set by pattern choice and SDB strength, becomes the dominant lever for long-run agent reliability as per-call model variance compresses with each model generation.
