# New Paper Imported — Architecture Patterns For Prod Llmagents

*Imported: 2026-05-20 16:10*

## Summary

**What it is:** The stochastic-deterministic boundary (SDB) is a four-part contract that specifies how a large language model (LLM) output becomes a system action in production LLM agents.
**Key contribution:** The proposed SDB methodology provides a framework for selecting and composing runtime architecture patterns for production LLM agents, allowing practitioners to design around the boundary explicitly.
**Main finding:** The application of the SDB methodology to five workloads demonstrates that architectural momentum, set by pattern choice and SDB strength, becomes the dominant lever for long-run agent reliability as per-call model variance compresses with each model generation.
