## Overview



<!-- bio-agent -->
Boxuan Li is a computer science researcher at Carnegie Mellon University whose work focuses on developing intelligent AI agent systems capable of complex human-like interactions with software environments. He is a core contributor to the OpenHands platform, a comprehensive open-source framework that enables AI agents to write code, interact with command-line interfaces, and browse the web through a modular event stream architecture and sandboxed runtime environments. His research involves designing multi-agent delegation systems and evaluation frameworks that assess agent performance across challenging tasks including software engineering benchmarks and web interaction scenarios. The OpenHands project, to which Li contributed significantly, has achieved substantial community adoption with over 2,100 contributions from 188 developers and includes an agent hub featuring numerous generalist and specialist agents. His work demonstrates the practical feasibility of building AI systems that can autonomously handle complex software development tasks while maintaining safety through containerized execution environments. Through this research, Li is advancing the broader vision of creating autonomous AI agents that can meaningfully contribute to software engineering and scientific research workflows.
