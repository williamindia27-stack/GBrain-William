## Overview



<!-- bio-agent -->
Yuexiang Xie is a computer scientist at Alibaba Group specializing in large language model systems and agentic application frameworks. He co-develops AgentScope, a developer-centric framework that abstracts foundational components including message, model, memory, and tool modules to enable flexible agent-environment interactions. His work implements the ReAct paradigm within AgentScope, enabling agents to combine explicit reasoning with tool use in iterative closed-loop execution cycles. The framework he helps architect delivers industrial-grade performance through parallel tool calls, asynchronous execution, and real-time steering capabilities, alongside integrated agents such as browser-use, deep research, and meta-planner variants. His research contributes to bridging the gap between academic agent development and practical production deployment by providing comprehensive toolkits, evaluation modules, and runtime sandboxes for safe, scalable agentic applications.
