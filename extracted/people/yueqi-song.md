## Overview



<!-- bio-agent -->
Yueqi Song is a computer science researcher at Carnegie Mellon University working on AI agents and software development. He is a core contributor to the OpenHands platform, a comprehensive system that enables large language models to write code, interact with command lines, and browse the web through a modular event stream architecture. His work encompasses the design of runtime environments with docker-sandboxed operating systems and the implementation of multi-agent delegation mechanisms that coordinate between generalist and specialist agents. The OpenHands platform, which has attracted over 2,100 community contributions, features evaluation frameworks across challenging tasks including software engineering benchmarks like SWE-BENCH and web interaction tasks like WEBARENA. Through this work, Song contributes to advancing the development of AI agents capable of performing complex, real-world interactions comparable to human developers, with applications spanning software engineering, web development, and scientific research.
